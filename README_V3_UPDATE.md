# LiveRP Admin Panel v3 — обновление

Что добавлено:

- исправлен скролл: страница теперь скроллится через обычный браузерный скролл, sidebar/topbar остаются удобными;
- добавлен раздел **Отделы / разделы**;
- owner/admin может создавать отделы и выбирать, какие колонки показывать: обзвоны, рапорты, обучения, активность, наказания;
- в составе администрации добавлены отделы, принятые рапорты и группировка по отделам;
- в системе повышений добавлены группировки по отделам и кнопки **Повысить** / **Понизить**;
- при наказании ранг/должность подтягивается из состава автоматически;
- сильно обновлён визуальный слой LiveRP: glass UI, новые карточки, фон, эффекты, анимации, улучшенные таблицы.

## Как обновить существующий проект

1. В Supabase откройте SQL Editor.
2. Выполните файл `liverp_update_v3.sql`.
3. На GitHub замените файлы:
   - `index.html`
   - `style.css`
   - `script.js`
   - `supabase.js`
   - `supabase.sql`
   - `README.md`
4. Добавьте файлы:
   - `liverp_update_v3.sql`
   - `README_V3_UPDATE.md`
5. Откройте сайт с обновлением кэша:

```text
https://rilavion.github.io/gta5rp-admin-panel/?v=40
```

6. Нажмите `Ctrl + F5`.

## Важно

Перед загрузкой проверьте `supabase.js`: в нём должен быть ваш реальный Supabase URL и `sb_publishable_...`, а не заглушки.

`sb_secret_...` / `service_role` в `supabase.js` не вставлять.
