# LiveRP Admin Panel v2 — что обновлено и как поставить

## Что изменено

- Брендинг заменён на **LiveRP | Admin Control Panel**.
- Добавлена вкладка **Руководство**.
- В составе администрации добавлены **ранги 1–11**, кастомная должность и отметка «Руководство».
- Старое поле «Должность» в интерфейсе заменено на **Ранг / должность**.
- Наказания автоматически подтягивают ранг/должность выбранного администратора из состава.
- В истории обзвонов появилась кнопка редактирования: можно позже добавить откат/видео, изменить комментарии, статус, тренера и пересчитать ответы.
- Вопросы считаются только по отвеченным вопросам.
- В конструктор вопросов добавлено управление кастомными категориями: переименование и отключение категории.
- В настройках добавлены 2 темы, фоновые эффекты, включение/отключение анимаций.

## Что такое «привязать аккаунт к записи в составе»

Пользователь — это аккаунт для входа на сайт: email, пароль и роль доступа.
Запись в составе — это карточка конкретного администратора: ник, Discord, ранг, отдел, наказания, выплаты и активность.

Привязка нужна, чтобы аккаунт входа был связан с конкретным человеком из таблицы состава.

## Обновление уже существующей базы

Если база уже была создана старым `supabase.sql`, сначала выполните в Supabase SQL Editor файл:

```text
liverp_update.sql
```

Он добавит новые поля в таблицу `admins`:

- `rank` — ранг от 1 до 11;
- `custom_position` — кастомная должность;
- `is_leadership` — отметка руководства.

Также он добавит настройки повышения по рангам 1–11.

## Обновление файлов на GitHub

Замените в репозитории файлы:

```text
index.html
style.css
script.js
supabase.js
README.md
```

Добавьте новый файл:

```text
liverp_update.sql
```

После загрузки откройте сайт с очисткой кэша:

```text
https://rilavion.github.io/gta5rp-admin-panel/?v=30
```

и нажмите `Ctrl + F5`.
